# html_advanced

Semantic HTML exercises for the Techium website project.

## Included files

- Progressive task files such as `0-index.html` through `39-styleguide.html`
- Final pages including `index.html`, `article.html`, `about.html`, `latest_news.html`, `contact.html`, and `styleguide.html`
- Supporting assets in `images/`, `favicon/`, `favicon.ico`, `favicon.png`, and `logo-black.png`
